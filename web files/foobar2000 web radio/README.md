# Foobar2000 Web Radio Interface

This setup is to be an online radio using a playlist in foobar2000

Software required is foobar2000 with the following components:

https://www.foobar2000.org/components/view/foo_out_upnp

https://www.foobar2000.org/components/view/foo_upnp

https://www.foobar2000.org/components/view/foo_youtube


Additionally, an http server like apache 2.4 is required to host the html page


By default this will only work on your local network, but can be expanded to the internet if you port forward your router
