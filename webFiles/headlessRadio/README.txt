This radio is designed to be used on a linux system completely from the terminal (headless)

Requirements:
- should be able to play youtube audio and local audio files (mp3, flac, etc)
- should have a web interface as the primary interaction (easy to see and change queue)
- youtube items should be treated equally as local items


Required software:
- webserver (apache recomended)
- php
- mplayer
- mps-youtube
- cmus?
